# FunnelIQ – E-commerce Conversion Analytics Dashboard

A product analytics dashboard built as part of my PM Intern application to Myntra.

## 🎯 Problem Statement
E-commerce platforms like Myntra lose significant revenue due to drop-offs at various stages of the conversion funnel. This tool helps PMs identify friction points, track KPIs, and generate AI-powered product recommendations.

## 📊 Features
- **Conversion Funnel Visualization** – Stage-by-stage drop-off analysis (Homepage → Order)
- **KPI Cards** – CVR, Bounce Rate, Avg Session Duration, AOV
- **Segment Filters** – Mobile, Desktop, New Users, Returning Users
- **AI Insights Engine** – Claude AI answers PM questions with data context
- **Segment Breakdown** – Conversion by device × user type
- **Category CVR** – Performance by product category

## 🛠️ Tech Stack
HTML, CSS, JavaScript
